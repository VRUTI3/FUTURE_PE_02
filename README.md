# FUTURE_PE_02
AI Prompt Engineering project that generates website copy and UGC ad content for local businesses using structured prompts.
## 📢 Task 2 – AI UGC Ad Generator

This task demonstrates how prompt engineering can generate authentic UGC-style marketing ads.

The system generates:

• multiple attention-grabbing hooks  
• short-form ad scripts  
• persuasive calls-to-action  
• social media captions  

These outputs are designed for platforms like Instagram Reels, TikTok, and YouTube Shorts.
